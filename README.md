# 📖 Overview

LanguageHub is a modern AI-powered multilingual translator built entirely using the Vanilla Web Stack (HTML, CSS, and JavaScript).

The application combines browser-native technologies with online translation services to provide fast and seamless translation of text and speech while maintaining a lightweight architecture.

Unlike traditional translator applications that require heavy frameworks, LanguageHub focuses on:

- ⚡ Fast Performance
- 🌍 Multi-language Translation
- 🎤 Voice Recognition
- 🔊 Natural Voice Playback
- 📱 Mobile-First Experience
- 💻 Progressive Web App Support

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/LanguageHub.git
```

Move into the project

```bash
cd LanguageHub
```

Run a local server

```bash
python -m http.server 8080
```

Open

```
http://localhost:8080
```

# 🔒 Privacy

LanguageHub does not permanently store translations on any server.

History and favorite phrases are stored locally in your browser using LocalStorage unless online translation services are used to process the text.
